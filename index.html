功能特性

🎯 方向键映射

将 CapsLock + h/j/k/l 映射为左/下/上/右方向键，无需离开主键盘区域即可快速导航，大幅提升操作效率。

⚡ 功能键映射

支持 Home/End、PageDown/PageUp 以及 F1-F12 等常用功能键的快捷访问，让常用键位触手可及。

🌍 多语言支持

基于物理按键位置（Scan Code）进行识别，完全不受键盘布局影响，支持所有语言和布局。

🖥️ 系统托盘

提供可视化的托盘图标，支持一键启用/禁用功能，右键菜单提供更多操作选项。

📝 日志调试

详细的运行日志记录，便于排查问题和调试，支持多级别日志输出。

🚀 开机启动

支持 Windows 开机自动运行，安装后即可享受无缝的键盘增强体验。

⚙️ 配置灵活

使用 JSON 配置文件管理所有映射设置，支持自定义键位映射和功能配置。

🔒 轻量高效

基于 C11 语言开发，占用资源极低，运行稳定流畅，不影响系统性能。

核心优势

🎨

Vim 风格设计

灵感来源于经典 Vim 编辑器的键位设计，为程序员和文本编辑用户提供熟悉的操作体验。

⚡

提升工作效率

无需频繁移动手到键盘边缘的方向键区域，减少手指移动距离，显著提升输入效率。

安装指南

步骤 1: 下载项目

从 GitHub 克隆项目仓库或下载最新版本的压缩包

git clone https://github.com/yourusername/PowerCapslock.git

步骤 2: 运行安装脚本

右键点击 install.bat，选择"以管理员身份运行"

程序将自动完成编译、安装并添加到开机启动项

步骤 3: 开始使用

安装完成后，程序会自动启动。按住 CapsLock + 映射键即可使用！

步骤 4: 自定义配置（可选）

编辑配置文件 %APPDATA%\PowerCapslock\config.json 来自定义键位映射
