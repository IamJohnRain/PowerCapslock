日常输入更顺手

一个 CapsLock，连接导航、功能键和语音输入

默认映射覆盖方向键、Home、End、PageUp、PageDown 和 F1-F12。需要个性化时，可以直接在配置中心新增、编辑或删除映射。

主键区完成光标移动

按住 CapsLock 配合 H/J/K/L，即可触发左、下、上、右方向键。

按键映射即时生效

保存配置后重新构建映射表，新增、编辑和删除无需重启程序。

模型路径即时检测

选择 SenseVoice 模型目录后立即校验并加载；不可用时直接给出原因。

本地离线语音输入

基于 sherpa-onnx 与 SenseVoice，语音识别在本机完成，不依赖云端上传。

日志路径可配置

日志目录可以在配置中心修改，重启后新日志写入新的保存位置。

系统托盘常驻

托盘菜单支持启用、禁用、查看日志、打开配置和退出程序。

可视化配置中心

日志目录、模型目录和快捷映射集中管理

配置保存在当前用户目录，适合日常使用、备份和迁移。模型目录支持选择外层 models、SenseVoice-Small 或最终模型目录。

1. 打开托盘菜单 右键 PowerCapslock 托盘图标，进入配置中心。
2. 选择目录 日志目录和模型目录都支持浏览按钮选择。
3. 保存配置 按键映射立即生效，模型路径立即校验。
4. 回到工作流 无需重启即可继续使用快捷键和语音输入。

本地离线语音输入

按住 CapsLock + A 说话，松开后自动输入文本

语音模型来自 FunAudioLLM/SenseVoice，模型文件单独发布。下载模型包后，在配置中心选择模型目录并保存，加载成功后即可使用。

  • 识别在本机完成，日常内容不上传云端。
  • 模型目录缺少 model.onnx 或 tokens.txt 时会明确提示。
  • 模型加载成功后会弹窗确认，无需重启程序。

快捷映射

默认 Vim 风格导航，也支持自定义映射

映射列表包含触发键、输出键和中文动作说明。新增表单会校验按键名称，避免重复触发键。

组合键 输出 常用场景
CapsLock + H/J/K/L 方向键 在编辑器、浏览器和输入框中移动光标。
CapsLock + I/O Home / End 快速跳到行首或行尾。
CapsLock + U/P PageDown / PageUp 浏览长文档和列表。
CapsLock + 1-0 F1-F10 访问编辑器、IDE 和系统快捷功能。
CapsLock + - / = F11 / F12 全屏、开发者工具和更多功能键场景。
CapsLock + A 语音输入 按住开始听写，松开后输入识别结果。

下载与使用

主程序和模型分开下载，目录更清爽

主程序包包含二进制产物、WebView2Loader 和运行时 DLL；模型包体积较大，单独下载后在配置中心选择模型目录。

1. 下载主程序 解压 PowerCapslock-v0.2.0-windows-x64.zip，运行 powercapslock.exe。
2. 打开配置中心 右键托盘图标，选择“配置”，设置日志目录、模型目录和按键映射。
3. 启用语音输入 下载模型包并选择目录，看到加载成功提示后即可使用 CapsLock + A。
