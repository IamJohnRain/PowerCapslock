# PowerCapslock v0.4.3.preview

这是一次聚焦截图 OCR 浮窗体验的预览版更新，重点修复了 OCR 挂起、拖动、划词选择与复制链路里的几处关键问题。

## 下载

- 主程序：`PowerCapslock-v0.4.3.preview-windows-x64.zip`

## 本次更新

- 恢复普通截图挂起能力，截图工具栏重新提供挂起按钮。
- 修复 OCR 挂起浮窗与普通挂起浮窗互相干扰的问题，截图可以再次正常移动。
- 修复 OCR 结果返回前后浮窗状态不同步的问题，避免点击 OCR 后没有挂起截图或一直停留在处理中。
- 修复 OCR 文字区域命中不稳定的问题，鼠标已经变成 I 光标时可以正常按下并拖动划词。
- 修复中文 OCR 文本复制链路中的间距处理问题，复制结果与中文连续文本更一致。
- 优化 OCR 选中高亮表现，选中词底色调整为 `RGB(153, 204, 255)`，拖拽时更容易看清当前选区。
