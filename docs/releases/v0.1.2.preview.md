# PowerCapslock v0.1.2.preview 发布说明

## 更新内容

### 新功能
- **语音输入功能**：新增离线语音输入支持，按 CapsLock+A 触发语音识别
- **语音输入提示窗口**：添加语音输入时的可视化提示窗口，显示录音状态
- **键位映射测试功能**：新增命令行测试工具，支持测试键位映射配置
- **命令行测试模式**：支持多种测试模式，包括语音测试、配置测试、键位映射测试等

### 修复与优化
- **修复语音输入中文乱码问题**：解决语音识别结果中文显示乱码的问题
- **修复松开快捷键无法停止录音的问题**：改进录音停止逻辑，确保按键释放时正确停止
- **修复音频采集和日志编码问题**：优化音频采集流程和日志编码处理
- **修正 I/O 键映射顺序**：将 CapsLock+I 映射为 Home，CapsLock+O 映射为 End
- **优化日志模块初始化**：修复日志模块初始化顺序问题，避免日志丢失
- **改进配置对话框**：优化配置对话框代码结构和用户体验

### 命令行工具增强
- `--voice-test` / `--dry-run`：语音测试模式，支持输出到文件
- `--test-audio-file <mp3> <expected>`：音频文件识别测试
- `--test-capslock-a [output]`：CapsLock+A 钩子测试
- `--test-key-mapping <from> <to> [output]`：键位映射测试
- `--test-config`：配置加载保存测试
- `--test-config-dialog`：配置对话框测试
- `--cleanup-logs`：日志清理测试
- `--enable-voice` / `--disable-voice`：命令行启用/禁用语音输入

## 安装说明

### 全新安装
1. 下载 `PowerCapslock-v0.1.2.preview.zip` 文件并解压
2. 运行 `install.bat` 进行安装（需要管理员权限）
3. 安装后程序将在系统托盘运行

### 语音输入功能启用
1. 首次运行时会询问是否启用语音输入功能
2. 如需启用语音输入，需要下载 SenseVoice 模型文件：
   - 下载地址：https://github.com/HaujetZhao/CapsWriter-Offline/releases/tag/models
   - 下载文件：`sherpa-onnx-sense-voice-zh-en-ja-ko-yue-2024-07-17.zip`
   - 解压到 `models/SenseVoice-Small/` 目录
   - 最终路径：`models/SenseVoice-Small/sherpa-onnx-sense-voice-zh-en-ja-ko-yue-2024-07-17/model.onnx`
3. 重启 PowerCapslock 即可使用语音输入功能

## 使用说明

### 基本功能
- **CapsLock 作为修饰键**：按下 CapsLock 后，配合其他键实现快捷操作
- **Vim 风格导航**：CapsLock + H/J/K/L 对应 ←/↓/↑/→
- **功能键映射**：CapsLock + 数字键对应 F1-F12

### 语音输入
- **触发方式**：按下 CapsLock+A 开始录音，松开停止录音
- **语音提示**：录音时显示提示窗口，显示录音状态
- **识别结果**：识别后的文本会自动输入到当前焦点位置

### 命令行测试
```bash
# 语音测试模式
PowerCapslock.exe --voice-test output.txt

# 键位映射测试
PowerCapslock.exe --test-key-mapping "h" "left" test-result.json

# 配置测试
PowerCapslock.exe --test-config
```
